# IMDB

* IDE used
    - Visual Studio Code 2017 (community edition)
    - Sql Server 2008 R2

* How to run the project 
    - Create a Database with name - IMDB and execute the IMDB.sql script.
    - Change the Server Name in the web.config file.
    - Hit F5 in Visual Studio.
